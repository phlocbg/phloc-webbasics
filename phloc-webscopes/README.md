#phloc-webscopes
Library for handling scopes in web applications.

------------------------------------------------------------

**Java 17 or higher** 

#Maven usage
Add the following to your pom.xml to use this artifact:
```
<dependency>
  <groupId>com.phloc</groupId>
  <artifactId>phloc-webscopes</artifactId>
  <version>7.0.1</version>
</dependency>
```
